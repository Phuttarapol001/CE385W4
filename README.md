# W4-1
เขียนฟังก์ชัน รับ พารามิเตอร์ 3ตัว คือ ตัว type, a, และb
"type" คือ ประเภทดำเนินการทางคณิตศาสตร์ เช่น add คือ บวก subtract คือลบ multiply คือคูณ divide คือหาร
a และb คือจำนวนที่ต้องนำมาคำนวณ

โดยใช้คำสั่ง if, else if และ else ในการตัดสินใจว่าจะใช้ประเภทการคำนวณอันไหน  และต้องคืนค่าผลลัพธ์ตามประเภท

สร้างฟังกฺชัน ที่ใช้ setTimeout เพื่อจำลองการทำงานที่ใช้เวลา 2วินาที และใช้ callback เพื่อส่งข้อมูลกลับ
สร้างฟังก์ชัน return Promise คืนค่า promise ที่ resolve .then() ใช้รับค่า      .catch() ใช้ดัก error

# W4-3

สร้างฟังก์ชันที่คืนค่าเป็น Promiseตรวจสอบเงื่อนไขภายใน Promiseใช้ async / await เรียกใช้งานดักจับ error ด้วย try / catch

ฟังก์ชันแรก ฟังก์ชัน return Promise รับค่า timeout (หน่วยเป็น millisec)

ใช้ setTimeout จำลองการทำงานแบบรอเวลา

ฟังก์ชันสอง เป็นฟังก์ชัน asyncใช้ await รอผลลัพธ์จาก Promiseใช้ try / catch เพื่อจัดการ error

# W4-4

เป็นฟังก์ชันที่ดึงข้อมูลจากหลายเซิฟเวอร์มาพร้อมกัน มีเซิร์ฟเวอร์หลายตัวบางตัวอาจล้มเหลว (reject)ขอแค่ มีอย่างน้อย 1 ตัวสำเร็จ ก็พอ

เริ่มเรียกทุก Promise พร้อมกันถ้ามี อย่างน้อย 1 Promise resolve.then() จะทำงานทันทีจะ ไม่สนใจ Promise ที่ reject  .catch() จะทำงาน ก็ต่อเมื่อทุก Promise reject หมด
